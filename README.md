# storiq

Organize inventory, find what matters, and hand off tasks clearly.

storiq is an inventory and task handoff app for people managing physical items across real-world spaces.

## Links

- Website: https://getstoriq.com
- Repository: https://github.com/nicholeproduces/storiq-site
- Privacy policy (for app / store listings): https://getstoriq.com/privacy (use your live site URL until DNS points here)

## What it does

- Capture and organize inventory across bins, rooms, and locations
- Find items quickly with searchable records and clear location details
- Create simple jobs and handoff lists from selected inventory items
- Share focused task context without exposing your full inventory
